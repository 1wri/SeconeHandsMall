## 一、项目介绍

语言: Java  数据库:MySQL

技术展示：

①：架构: B/S、MVC

②：系统环境：Windows/Mac

③：开发环境：IDEA、JDK1.8、Maven、Mysql8.0

④：技术栈：Java、Mysql、SpringBoot、Mybatis、Vue

项目功能：

前台：

1.首页（全部查询、分类查询、关键字搜索）

2.发布物品

3.查看消息

4.个人中心（编辑个人信息、编辑收货地址）

5.状态查询（我发布的、我下架的、我收藏的、我卖出的、我买到的）

后台：

1.交易物品管理（上线的二手物品（可以违规下架）、下架的二手物品（可以永久删除））

2.订单信息管理（删除）

3:用户信息管理（正常用户（可封号）、违规用户（可以解封）、管理员（可添加管理员））
